**[Prefer Set Instead of List in @ManyToMany Relationships](https://github.com/AnghelLeonard/Hibernate-SpringBoot/tree/master/HibernateSpringBootManyToManyBidirectionalListVsSet)**

**Description:** This is a Spring Boot example of removing rows in case of a unidirectional/bidirectional `@ManyToMany` using a `List` and a `Set`. The conclusion is that `Set` is much better!

**Key points:**\
     - using `Set` is much more efficent than `List`
